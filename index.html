AERO SAE 2023-2024

Developing avionics software, embedded software, and robotics for the Aero SAE 2024 competition, ensuring the successful launch of an autonomous secondary aircraft to deliver payloads to specified targets. Programming computer vision with Raspberry Pi, Python, OpenCV to detect 100% of targets for no GPS section of competition.

Duality

The goal of Duaity was to make an automated monoblock pill bottle filling system for small batch manufacturers. Programmed and wired an Arduino microcontroller in low-level C++ to implement precise break sensor beam detection and simultaneous control of stepper motors, Duality increased performance by 454% compared to manually filling the bottles while maintaining 99% accuracy.

Sprite Brawl

The 2D 1v1 Javascript fighting game with fully implement physics and unique attacks that has online functionality to connect users. The project utilizes a nodeJs backend with a Firebase database, and a websocket connection for a low-latency gaming experience.
